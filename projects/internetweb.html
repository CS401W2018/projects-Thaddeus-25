Internet

A system that allows computer networks to communicate with each other using set of protocols. The internet enable users to access WWW (World Wide Web)


Web

An information system (part of the internet) that allow users to share their website or pages


How are the Internet and Web different?

The internet is the network that connects devices to one another, but the web is a system that runs in the internet that shows user collections of pages, images, etc.


ASCII and Unicode

ASCII and Unicode are both used to encode text on computers and other devices. ASCII is an older encoder that only supports a limited number of characters, but Unicode is able to encode more characters including text from different languages.


What is UTF-8

UTF-8 is the most common character encoder that is used by the World Wide Web.
