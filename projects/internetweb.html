Internet

A system that allows computer networks to communicate with each other using set of protocols. The internet enable users to access WWW (World Wide Web)


Web

An information system (part of the internet) that allow users to share their website or pages


How are the Internet and Web different?

The internet is the network that connects devices to one another, but the web is a system that runs in the internet that shows user collections of pages, images, etc.


ASCII and Unicode

ASCII and Unicode are both used to encode text on computers and other devices. ASCII is an older encoder that only supports a limited number of characters, but Unicode is able to encode more characters including text from different languages.


What is UTF-8

UTF-8 is the most common character encoder that is used by the World Wide Web (WWW). Each character is represented by 1-4 bytes.


URL

URL stands for "uniform resource locator" Which is the address where a webpage or file is on the internet.


Domain Name

Domain name is the name of a website. This enable users to search a website easily by typing in the domain's name instead of the IP Address.


HTTP

HTTP (Hypertext Transfer Protocol) is a protocol that is used to transfer data on the internet. It is used to transfer data between a server and a client.


HTTPS

HTTPS is the same as HTTP but it transfers data in a more secure connection, by establishing an encrypted connection before transferring data.


IP Address

IP (Internet Protocol) Address is a unique number that is given to each device that is connected to the internet.


DNS

DNS or Domain Name System is used to translate domain names into IP Address, that allows users to have access to websites.


W3C Recommendations

W3C Recommendation is a standard or guideline to ensure that a web is stable and compatible for all platforms.


XML, XHTML, HTML, HTML5

These are all markup languages, which means they are used to write/construct text and other contents on a webpage. XML is mostly used for data transfers. XHTML is more like a XML-based version of HTML. HTML is designed to create the structure of a document. HTML5 is the next version of HTML, which is more flexible and compatible with all devices.
