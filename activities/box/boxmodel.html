A block tag will show up on the screen with the default width covering the whole screen and the height depends on the text it contains (for example div, header, nav, h1-h6, p, ul, etc), it also creates a new line. While Inline text only occupy the space it given without making a new line (span, em, a, img, etc).

2. margin is the space outside the box between the border and other tags, padding is the space inside the box between the content with the border (it adjust the distance of the content by pushing the content), border is a line that outlines the tag.

3. Fixed size like px, thin, in, cm, etc. and also relative size like % (percentage), em, etc.
