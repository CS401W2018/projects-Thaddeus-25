Engagement

1. Inline CSS: is when the style (CSS) is written inside a tag.
Embedded CSS: is when the style (CSS) is written in the same file as the html, most commonly in the head tag.
External CSS: is when the style (CSS) is written on another file, which is a CSS file.

2. So that it is not restricted for the web designers to work (for example if there is an inline CSS on the file, whatever the web designer does to that specific tag won't work), and so we don't have to make multiple CSS styles (1 CSS file is enough for multiple HTMLs).

3. A selector is used to pick/choose specific tags, class, and id that we want the CSS to work on.
